# SistemaGestorPro
Sistema de gestão e analise documental de fornecedores em Laravel. O GestorPro é um sistema desenvolvido com o objetivo de otimizar o gerenciamento de fornecedores, 
centralizando o controle de documentos, validações e processos de aprovação em uma única plataforma. As integrantes da equipe são Amanda Lemos, Kéren Hapuque e Maria Eduarda Mesquita.

## Objetivo
O sistema tem como principal finalidade organizar e automatizar a gestão documental de fornecedores, garantindo maior eficiência, segurança e conformidade nos processos empresariais, esse sistema é
voltado para empresas e fornecedores que necessitam de um controle mais rigoroso e estruturado sobre documentação, contratos e regularidade cadastral.

#### Modelo Entidade-Relacionamento (MER) 
Representa a estrutura do banco de dados do sistema GestorPro, evidenciando as principais entidades, seus atributos e os relacionamentos entre elas. Através desse modelo, é possível compreender como as informações são organizadas e conectadas, como por exemplo a relação entre fornecedores, documentos e usuários do sistema. O MER foi projetado com o objetivo de garantir consistência, integridade dos dados e facilitar operações como cadastro, consulta, atualização e controle de documentos, além de permitir a rastreabilidade das informações por meio de históricos e auditorias. Esse modelo serve como base para a implementação do banco de dados, contribuindo para uma estrutura organizada, escalável e alinhada às necessidades do sistema.
